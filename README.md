# 🎧 Aurix Headset

Projeto de uma page moderna para apresentação do headset fictício **Aurix**, com foco em experiência do usuário (UX), interface visual (UI) e animações avançadas.

---

## 🚀 Objetivo

Desenvolver uma página imersiva que simule a apresentação de um produto premium, utilizando interações, motion design e elementos visuais de alto impacto.

---

## ✨ Funcionalidades

* 🎥 **Hero Section com Vídeo**

  * Apresentação inicial dinâmica e envolvente

* 🔄 **Carrossel Interativo**

  * Navegação fluida entre características do produto

* 🔍 **Visualização 360°**

  * Interação direta com o produto
  * Experiência mais realista

* 🎬 **Animações com GSAP**

  * Transições suaves e profissionais
  * Controle avançado de timing e movimento
  * Maior sensação de fluidez e qualidade premium

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* GSAP (GreenSock Animation Platform)

---

## 🎯 Foco do Projeto

* Experiência do usuário (UX)
* Interface moderna (UI)
* Motion design aplicado ao front-end
* Simulação de páginas de alto nível (Awwwards)
